8. KEY FINDINGS
------------------------------------------------------------------------
  a) Both LSTM and XGBoost achieve ~99.9% accuracy because the
     20 hand-crafted features (syn_burst_ratio, failed_conns,
     lb_syn_60s, etc.) are direct, near-perfect discriminators
     of scanning activity at each individual timestep.

  b) Temporal context adds minimal value for per-timestep
     classification in this dataset. The model can classify
     correctly even with a 40-second window (SEQ_LEN=4).

  c) The LSTM adds genuine value over XGBoost for:
     - Early onset detection (flags scans within 10-20 s)
     - Temporal smoothing (no jitter between consecutive predictions)
     - Graceful degradation under missing/noisy features

  d) For a production dashboard:
     - XGBoost is simpler, explainable, and equally accurate
     - LSTM is worthwhile if early detection latency is the primary KPI
     - SEQ_LEN=8 (80 s) balances accuracy and alert delay
